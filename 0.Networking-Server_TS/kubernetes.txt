K8s Failures
1.Image pull backoff : pull container image for prod
2.Crash loop bacloff : container inside pod crashes > k8s cannot run the app > config issues in pod.yml
3.Exit code 1 : Occurs when the container in a pod exists with non zero status code > container encounter error unable to start
4.Not ready : Occurs when node is not ready to recive the pod
